Project information

  • Name: Pi Race Strategy
  • Client: Cosworth Electronics
  • Project Date: 2018
  • Project Site: N/A
  • Tech Stack: .NET Framework, WPF
  • Additional Information: This project was developed as an internal testing tool and demo product for potential customers of Cosworth's bespoke SDK (Pi Data Access).

    This was built in .NET Framework using Microsoft WPF utilizing a MVVM design pattern, I was the sole developer on this project.

    This Windows application connected to a local COM port, attached to a Real-time data stream from Cosworth's SDK, displayed data and performed real-time calculations to calculate the potential pit stop window and strategies. A lot of clients built many in-house products using this as a foundation.
